Notes & Summaries

Knowledge base and learning resources

The AZ-900: Microsoft Azure Fundamentals is an entry-level certification that validates basic cloud computing knowledge and familiarity with Microsoft Azure services. It covers cloud concepts, core Azure services (e.g., virtual machines, storage), security, pricing, and support. Ideal for beginners, it requires no prerequisites, involves a 40–60 question exam, and costs ~$99 USD. It’s a great starting point for Azure certifications and cloud careers.

The CompTIA Security+ SY0-701 is a globally recognized, vendor-neutral certification validating core cybersecurity skills for IT professionals. Launched November 7, 2023, it covers five domains: General Security Concepts (12%), Threats/Vulnerabilities/Attacks/Mitigations (22%), Security Architecture (18%), Security Operations (28%), and Security Program Management (20%), with emphasis on zero trust, cloud security, and hybrid environments. The exam features up to 90 multiple-choice and performance-based questions, lasts 90 minutes, requires a 750/900 passing score, and costs ~$392 USD. No prerequisites, but 2 years of IT experience is recommended; it's ideal for entry-level roles like security analyst and a foundation for advanced certs.
